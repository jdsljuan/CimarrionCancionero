¿Cómo usar la app?

Pantalla Principal (Izquierda)

Está pantalla es la más importante pues es donde las SPA serán mostradas. Por defecto la aplicación se dirige a cargar está página pero puedes cambiarla en la pantalla de controles.

Pantalla de Carga (Derecha)

La pantalla de carga se mostrará cada vez que un vínculo nuevo sea cargado. La barra de carga que se muestra en ella muestra el tiempo de carga del vínculo una vez cargado como un Toast en la pantalla principal. Si la barra llega a su límite de dos (2) segundos, mostrara el color amarillo y si el tiempo de carga sobrepasa los cuatro (4) segundos se pondra roja.



Navegación

Abre está pantalla presionando la tecla ATRÁS. En está pantalla puedes: Salir de la aplicación de manera definitiva, cambia de SPA escribiendo su dirección en la barra de texto, luego presiona URL, y por último, si deseas cargar mi SPA presiona APP.



Usar el App como un Navegador Web

Si sales del dominio de la aplicación y usarás este aplicativo como navegador web, la barra de estado de Android se tornará roja, de lo contrario será siempre negra como se muestra en la imagen. Recuerda que debes conocer la URL completa del aplicativo que deseas visitar.
